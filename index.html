About the Project

Blind Assist VR is a low-cost, privacy-respecting wearable designed to give blind and visually-impaired users audio understanding of the world in front of them. It combines a camera with efficient on-device AI models, an ultrasonic sensor for obstacle distance, offline speech for reliability, and optional cloud features when Wi-Fi is available.

Why it matters

Real-time guidance reduces collisions, improves independence, and supports safer navigation indoors and outdoors.

Built for budget

Runs on Raspberry Pi 3 (1 GB). Uses quantized models, smart filtering, and sensor fusion to keep accuracy high without extra hardware.

Privacy first

Offline detection and voice processing by default. Optional cloud only when the user chooses.

Core Features

Modern assistive capabilities today, with a roadmap to advanced features.

Object Detection

On-device TFLite model detects common objects and announces them. Confidence-based smoothing avoids false positives.

Obstacle Warnings

Ultrasonic sensor alerts when objects are within 30 cm. Adjustable threshold and haptic option.

Offline Voice

Vosk STT understands commands like “What’s in front of me?”; pyttsx3 speaks back instantly.

Face Recognition

Save trusted faces locally and identify them later with tunable thresholds.

SOS & Family

Planned: voice/physical SOS sends photo + location to family app with push notifications.

Home Automation

Planned: control lights and appliances via MQTT or smart-home APIs.

Hardware

Primary components for the MVP build. All parts fit in a VR-style shell with ventilation and easy charging.

Pi

Raspberry Pi 3

1 GB RAM. Runs quantized models with CPU. Optional upgrade to Pi 4/5 later.

Cam

Camera

Pi Camera v2 or USB cam at 640×480 for speed; configurable frame rate.

US

Ultrasonic

HC-SR04 for distance alerts; tuned to ~30 cm threshold.

Mic

Audio I/O

USB/I2S mic + small speaker or bone-conduction headset.

BTN

SOS Button

Physical emergency trigger with debounce logic.

OPT

Future Upgrades

GPS, Coral USB, LiDAR, IMU, higher-res camera, battery monitor.

Roadmap

Clear phases so the most important features land first.

Phase 1 — Offline MVP

Object detection, ultrasonic warnings, offline voice, basic face save/recognize.

Phase 2 — Safety & Family

SOS with photo + location, simple family app, optional cloud Q&A.

Phase 3 — Expansion

OCR text reading, scene description, home automation, navigation assistance.
